# Bloom
A modern productivity web application that helps users organize their daily life through task management, habit tracking, and progress visualization.
Bloom is designed with a clean and calming interface inspired by nature, making productivity feel more enjoyable while helping users stay focused on their goals.

---

## Features
Task Management
- Create, edit, and delete tasks
- Mark tasks as completed
- Task completion statistics
- Beautiful and responsive task cards

Habit Tracker
- Create daily habits
- Track habit completion
- Visual habit progress
- Daily streak monitoring

Dashboard
- Overview of completed tasks
- Total task statistics
- Bloom Progress visualization
- Recent activity section
- Productivity summary

Theme Support
- Light Mode
- Dark Mode
- Smooth theme switching

Multi Language
- English
- Indonesian

Authentication
- User Registration
- Secure Login
- Forgot Password
- User session management
- Powered by Supabase Authentication
